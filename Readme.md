# Cancer Awareness Website

A simple, impactful website to spread awareness, educate people, and inspire hope.

## About the Project

This website is designed to spread cancer awareness, share important facts, give support resources, and encourage people to take early action.

Built with a clean UI, responsive layout, and soft emotional theme.
